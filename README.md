# AganithaTestRepo

The Paste Bin Project is developed enitirely usng node and expressjs and the persistence is achieved using Mongo DB 

The project is deployed in vercel and can be accessed using ::: https://aganitha-test-repo.vercel.app/

To run the project localy 

1: clone the repo
2: use npm i
3: in public/webpage.html replace the vercel domain with localhost:3000
4: use node server

after this you can access by just using localhost:3000 in any browser

For setting up the codebase i used mvc architecture to make it scalable and robust and introduced 3 different endpoints to perform the required tasks
